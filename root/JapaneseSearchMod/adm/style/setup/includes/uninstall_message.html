アンインストール

JapaneseSearchMod をアンインストールするには、

  1. AdminCP の “検索設定” ページ の “検索バックエンド” を “Fulltext native ja” 以外に設定し、
  2. JapaneseSearchMod によって変更されている以下のファイル内のコードを元のコードに戻し、
    • root/search.php
    • root/viewtopic.php
  3. 以下のディレクトリ及びファイルを手動で削除し、
    • root/includes/hooks/hook_JapaneseSearchMod.php - ファイル
    • root/includes/search/fulltext_native_ja.php - ファイル
    • root/JapaneseSearchMod - ディレクトリ
  4. AdminCP で “キャッシュの消去” を行ってください。
